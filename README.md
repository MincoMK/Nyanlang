# 냥랭 (Nyanlang)

세상에서 제일 귀여운 언어다냥!  
당연히 Brainfxxk 기반인건 알고 가라냥!

아래는 비교 표다냥.


| Brainfxxk | Nyanlang | Description               |
|-----------|----------|---------------------------|
| >         | ?        | 포인터 증가                    |
| <         | !        | 포인터 감소                    |
| +         | 냥        | 포인터가 가리키는 값 증가            |
| -         | 냐        | 포인터가 가리키는 값 감소            |
| .         | .        | 포인터가 가리키는 값 출력            |
| ,         | ,        | 포인터가 가리키는 값 입력            |
| [         | ~        | 포인터가 가리키는 값이 0이 아닐 때까지 반복 |
| ]         | -        | 포인터가 가리키는 값이 0이 될 때까지 반복  |
